七期寶璽榮耀世紀 角間大地坪電梯別墅

建案名稱: 【七期寶璽榮耀世紀 角間大地坪電梯別墅】

地址 : 台中市南屯區大富街 總價 : 5,880 萬
建坪 : 146.09 坪 地坪 : 66.25 坪
格局 : 5房 3廳 5衛 主建 : 97.71 坪
附屬 : 14.26 坪
樓層 : B1樓 / 4樓 完工 : 93 年 9月 類別 : 別墅/住家 車位 : B1-三車位

物件特色
1. 位七期南側低密度別墅區、森林公園、靜謚舒適・ 2. 惠文完全學區，附近都是高級豪墅，為七期最佳生活圈！ 3. 精美裝潢・生活機能便利・臨路角間電梯別墅・稀有釋出。 4. 寶璽建設七期品牌建商・南七期豪墅區・入則靜，出則繁・ 5. 本社區戶數單純，環境非常優雅寧靜。室內面積大共有4間套房，！適合三代同堂！ 6. 獨棟雙併 角間別墅…採光通風佳，屋況保持良好！鬧中取靜・ 7. 惠文完全學區・角間採光通風佳・四套房規劃・ 8. 側院的花園適合種植花花草草…有毛小孩也適合・ 9. 附裝潢，動線流暢，格局方正採光佳・

10.三面採光+屋主千萬裝潢超級物超所值・

11.臨近公益路商圈和黎明商圈…生活機能好…而且更是鬧中取靜・ 12.學區是明星學校…惠文雙學區,走路5分鐘到學校・ 13.交通便利上國道或中彰都方便・ 14. 屋主誠意出售，歡迎出價誠談・ 15. 空間規劃 B1視聽室+儲物間+3個平面大車位 1F~客廳+餐廳+廚房空間+1衛浴 2F~2間套房都有陽台 3F~2間套房都有陽台 4F~書房+陽光露台+洗衣工作陽台+多功能空間

社區介紹 「寶璽榮耀世紀」位於台中市大墩十二街，基地面積930坪。榮耀世紀由寶璽建設興建，於2004年9月竣工。 「寶璽榮耀世紀」的樓層規劃為地上4層、地下1層，地坪42～70坪，建坪102～175坪，格局規劃為5房2廳4衛，總戶數為20戶。 「寶璽榮耀世紀」的主結構為RC鋼筋混凝土，結合鋼筋極佳的抗拉性與混凝土極佳的抗壓性，讓樓宇在遇強風及地震時有側面位移小的特性，讓人在室內感覺不明顯，是為台灣目前最普遍也是技術最成熟的建築工法。 「寶璽榮耀世紀」外觀走現代古典建築風格，以紐約風格為主軸，搭配列柱、深凹長窗、立體花園露台、夜間燈光的規劃，在七期重劃區中是相當具有特色的樓宇。 「寶璽榮耀世紀」選用高格規的建材，包括德國衛浴、龍頭、按摩淋浴柱、義大利百萬廚具、電烤箱、洗碗機、日本烘碗機、健康能量活水器等，窗戶採日本YKK鋁門窗及換氣框，為住戶打造舒適的居家空間。 「寶璽榮耀世紀」在社區規為上運用全區植栽、綠化工程，特設15公分厚的外牆、深窗及低檯度開窗，而有節能和調節日照的效果。榮耀世紀的基地3面臨路，透過數位單晶片錄影機和45座CCTV監視系統，再加上安全防爆門，為住戶的住家安全把關。 「寶璽榮耀世紀」鄰近熱鬧的公益路，沿線商家林立，有便於住戶的生活採買，鄰近明星學區惠文國小、惠文高中而具有豐富的教育資源，榮耀世紀擁有多重的機能，讓住戶的生活更趨便利。 增值潛力 「寶璽榮耀世紀」位在台中市大墩十二街，是為該區域的精華地段。榮耀世紀外觀走現古代典風格，外部列柱、長窗、露台的設計，優美不在話下，榮耀世紀內部的建材採用高品質的設備，和多座監視系統，致力打造舒適安全的住家空間，為樓宇的附加價值往上加分。榮耀世紀全區以綠建築的概念設計，進而達到節能的效果，保值性十足。 「寶璽榮耀世紀」鄰近熱鬧的公益路，沿線生活機能發達，值得嘉許，而生活機制是多數在購屋前的一大考量，榮耀世紀擁有這樣的優勢，當然極具保值身價。榮耀世紀位在最近有百貨業者進駐的七期重劃區內，因商機的導入，帶動周邊房價起飛，榮耀世紀受其影響也水漲船高。 「寶璽榮耀世紀」擁高速公路、快速道路、高鐵、台鐵及轉運站等交通之便，而具有交通機能的優勢，人潮車潮的行經帶動錢潮的湧進，成為榮耀世紀增值的條件，前景看好。 「寶璽榮耀世紀」共20戶地上4層/地下1層2004-09竣工
