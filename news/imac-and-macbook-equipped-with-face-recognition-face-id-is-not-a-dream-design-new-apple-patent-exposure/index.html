iMac和MacBook搭載Face ID臉部辨識不是夢，蘋果專利曝光新設計

蘋果替 iPhone X 率先引入 Face ID 臉部辨識功能，這項辨識技術也延伸到 iPad Pro 產品上，不過瀏海造型依舊讓不少用戶無法直視，但有沒有可能會讓 Face ID 在 Mac 電腦上出現？這是很有可能發生的！蘋果分別在 2017 年iMac和 2020 年MacBook通過 Face ID 專利已經正式通過，在未來也將會看見 Mac 電腦搭載臉部辨識並非夢想。

根據外媒 patentlyapple 從美國專利商標局最新文件中發現，蘋果最近通過一份名為「用於電腦設備的光線辨識模組」專利文件，內容顯示能將 Face ID 功能加入至 iMac 和 MacBook 上，會在螢幕頂端崁入類似 iPhone 瀏海設計，也表示 iPhone 和 iPad Pro 上的 TrueDepth 相機模組，未來也將會移植到 Mac 電腦上，並且取代原本 Touch ID 指紋辨識功能。

專利文件中顯示，電腦設備的最新發展能讓用戶執行複雜功能，像是 Internet 瀏覽、聊天、設計、剪輯等。 Face ID 技術可結合在電腦設備，可透過紅外線辨識來識別用戶，其中文件還提及，透過 Face ID 也能夠辨識手勢和臉部情緒，可運用在聊天、產生表情符號、加密、解鎖等功能。

在專利文件中，也能夠看見 Face ID 被放置於 iMac 和 MacBook 螢幕頂端區塊中，透過專利圖片可得知，蘋果預設構想類似 iPhone 瀏海造型；如果要考慮到 iMac 和 MacBook 螢幕外型設計，想要將 Face ID 藏匿在螢幕黑條內也並非難事，畢竟空間非常足夠能容納 Face ID 辨識元件。

另外專利圖也顯示 iMac 背面透視圖，透露 Face ID 設計元件包含各種零件，包含有 ESD接地彈簧、麥克風、指示燈、RGB相機、環境光傳感器、支架零件、發光器、光檢測器、控制器等。

這份專利申請文件編號為 20200097747，蘋果從 2019 年第三季就已經提交，直到近日才正式通過，目前還尚未清楚蘋果打算在何時替 iMac 和 MacBook 加入 Face ID 辨識技術。

但是能夠讓筆電採用臉部辨識技術也並不是一個非常奇特的想法或技術，微軟很早已經能夠讓 Windows 筆電系統支援 Windows Hello，只要筆電開啟螢幕瞬間，就能開始辨識人臉，減少用戶輸入密碼流程，目前也只有 MacBook 能夠透過 Touch ID 指紋辨識來認證和支付 Apple Pay ，iMac 上還無法擁有 Touch ID 功能，相信蘋果不想落後對手太多，近幾年可能也會考慮替 iMac 或 MacBook 搭載 Face ID 臉部辨識功能。

洩露玄機？ 2020 年版 iMac 、 Mac mini 即將發表？！

坊間一直傳聞 Apple 會在 3 月舉行春季產品發表會，不過由於新冠肺炎疫情關係，實體發表會很大機會要取消。不過雖然沒有發表會，產品還是要賣的。去年一位神秘人 CoinX 率先在 Twitter 發表 iPhone 11 將以「 Pro 」來命名，上星期他又發功，發了兩則帖子，不過預示 iPad 會換上較高質素的新鏡頭，還表示 iMac 及 Mac mini 即將發表 2020 年版本！

其實 Mac mini 推出新版本並不奇怪，因為 Mac mini 對上一個版本已經是 2018 年 11 月的產品，所用的仍然是 Intel 8 代 CPU ，而標準配備的 SSD 容量就只有 128GB 。以去年 MacBook Pro 容量翻倍的做法，新 Mac mini 的標準容量很有機會由 256GB 起跳，而 CPU 方面，由於 Intel 還未推出與上一代型號所用的 i5-8400B 6 核 CPU 相應的 10 代 CPU ，相信新的 Mac mini 應該只會用上第 9 代 CPU 吧。

至於 iMac 就有點來得急，因為對上一次更新只是 2019 年 3 月，相信今次更新除了將全線型號的 CPU 升至 9 代之外，同時也會提升一下顯示卡吧。

蘋果設計之魂 Jony Ive 離開有什麼影響？

© Getty Images

與賈伯斯一手打造蘋果王國的蘋果前設計副總裁 Jony Ive，在去年宣告離職，為市場投下一顆震撼彈。Jony Ive 是當年蘋果的靈魂人物賈伯斯二次回歸的重要合作夥伴。當年賈伯斯二次回歸蘋果公司時，公司正在經歷財務危機，而直到 Jony Ive 與賈伯斯聯手打造出 iMac 和 iPod，從此改變了蘋果的命運，而他也在 2011 年獲得 KBE 勳銜成為爵士，可想而知他的重要性！今年推出的蘋果產品就已經不屬於Jony Ive的作品了，會有什麼樣的改變相當值得關注。

iMac

在去年推出 2019 更新版本的 iMac，有著高達 8 核心的 Intel 第九代處理器結合 Retina 顯示器，讓使用者處理工作更為高效，並且首次帶來 Radeon Pro Vega 繪圖處理功能，讓全新 iMac 處理繪圖效能快達 80%，非常驚人！而 2019 更新版的 iMac 有 21.5 吋和 27 吋兩種尺寸。21.5 吋 iMac 現在配備第八代 4 核心處理器，並首度加入 6 核心處理器可供選擇，可提供快達 60% 的效能；而 27 吋iMac 則是首度使用最高達第九代 6 核心和 8 核心的處理器，效能處理高達 2.4 倍，兩種尺寸都加強效能呈現！

iPod Touch

2015 年 7 月之後，iPod Touch 已經近四年都沒有更新了。但是在去年，Apple 突襲發佈了 iPod Touch 的 2019 版本，除了帶來新的顏色外，也為 iPod Touch 更新了 AR 實境擴增的功能，加上與iPhone 7 一樣的 A10 Fushion 晶片，讓使用者可以有更好的遊戲娛樂體驗！

iPad

蘋果日前帶來全新的iPad Pro，後面1200萬畫素超廣角鏡頭和1000萬畫素的廣角鏡頭搭配為AR技術準備的光學雷達掃描儀，功能更加強大了！同時這版iPad Pro還有搭上了觸控式軌跡板，讓iPad使用上更方便，更像是筆電的操作模式！

iPhone

Apple 預計在今年推出新一代的 iPhone SE 和 iPhone 12 兩種型號，繼去年推出的 iPhone 11 所搭配的三鏡頭設計後，全世界所有果迷都在熱烈討論接下來的 iPhone 會有什麼新的設計。今年預計推出的 iPhone SE 2 外型與硬體規格都類似 iPhone 8，擁有 4.7 寸 LCD 螢幕，太空灰、銀色和紅色三種配色選擇，但將會搭載蘋果最新的 A13 處理器和單鏡頭，同時還會有許多人相當懷念的實體 Home鍵和隨之搭配的 TouchID！而 iPhone 12 最受人注目的絕對就是搭載 4 鏡頭和支援 5G、恢復 iPhone 4 方正形狀這三件事，以及準備推出的全新「海軍藍」配色！

至於先前風評兩極的 iPhone 11 三鏡頭設計，儘管蘋果在發表會中相當強調這台相機的效能，提供給好幾位專業攝影師拍攝讓大家可以看見其驚人的效果，加上特別強調的夜攝功能，但是時下許多其他廠牌也都有相近或相同的功能，究竟 iPhone 的畫面能不能滿足你的需求，那便見仁見智了！

AirPods

AirPods Pro 自從推出之後獲得許多果粉的喜愛，不過 AirPods Pro 入耳式的設計和主打的降噪功能也讓它的售價比起前幾代要高上不少。最近蘋果就據稱正在開發「入門款式」的 AirPods Pro Lite！目前傳出的消息指出這個入門版本可能會是價錢更低，不過也不會降低主打的降噪性能，有可能會以縮小目前較大的體積為主，最新消息更指出這個版本將會是「耳罩式耳機」確切究竟會從哪裡進行精簡還有待後續的消息更新！

AirPods 屬於劃時代的產品，改變了許多人平時使用耳機的習慣！但是除了基本功能之外，是否還有一些「隱藏版」的功能是你不知道的呢？而市面上除了 AirPods 之外，還有 Powerbeats Pro 等藍牙耳機，到底要如何選擇最適合自己的耳機呢？不用擔心，我們也都為各位整理好了這些相關資訊，讓各位可以不再有後顧之憂！
