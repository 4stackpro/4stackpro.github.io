自動駕駛進化？這台 Model 3 可以看懂紅綠燈了

特斯拉又變得更聰明了嗎？外媒今日提供一段影片，顯示 Model 3 可以識別前方號誌並停車，或許離真正的自動駕駛又更進一步了。

在影片中，這台 Model 3 可以成功識別路段上的所有紅綠燈，仔細看螢幕可以發現，它還能夠辨認不同車道使用的號誌燈，最後它確認了前方的紅燈，並自己停下來。

#Tesla #Autopilot stopping for red lights for the first time. pic.twitter.com/M0zkiqyypl — Out of Spec Motoring (@Out_of_Spec) March 26, 2020

根據用戶反饋，它不僅能夠辨識紅綠燈，也可以認出 STOP 標誌，換句話說，這讓特斯拉在一般道路上自動駕駛又更進一步了。

我們肯定會問：怎麼樣才能啟用這個新功能？這在台灣有搞頭嗎？

可惜的是，這個功能並非官方公告的更新，而是被列為「即將推出」的功能。影片中的車主也未透漏他如何取得這次更新，想要嘗新的朋友，還得等一等。相信我，這種安全攸關的功能，還是多給點耐心吧。

▲ 對交通號誌做出反應，是特斯拉官方列為「即將推出」的功能。（Source：特斯拉）

第二個問題是，自動駕駛能克服台灣的街頭嗎？相信很多車主都對台灣路況充滿信心，自動駕駛如果能在台灣馬路上開車，應該全世界都沒問題了。事實上，台灣交通也不算是全世界最瘋狂的，但相對於美國來說，確實難度高了很多。

不過單就號誌辨識這一點，台灣大概不算太差，畢竟我們一個路口就有 7、8 個紅綠燈，想要錯過都難啊。

TESLA Model3真的漲價了！全車系漲5～6萬元

今天早上在台灣特斯拉相關社團最熱鬧的新聞就是在全球狂銷熱賣的 TESLA Model 3 不但沒有因為產量大攤平研發成本而降價，反而逆勢調漲了五～六萬台幣，雖說 Model 3 在國外早就漲價一波了，台灣區的價格倒是聞風不動很久，馬斯克嘴上也一直叫消費者「要快買，真的會漲價」，但由於之前有無預警大幅降價的紀錄所以大部分的消費者總是持觀望態度，因為正常來說哪有東西在正常狀態下會越賣越貴的？不過特斯拉剛好就是一個特例。
