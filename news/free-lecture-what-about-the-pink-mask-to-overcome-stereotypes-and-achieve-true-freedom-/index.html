粉紅色不好嗎？看完這些 Pink Tesla 也許你會改觀

每個人都有自己喜歡的顏色，別人身上穿的用的戴的顏色，都不應該被歧視，一起來看看這些熱愛粉紅色的特斯拉車主愛車，讓他們成為真正的「特粉」。

特斯拉原廠能選擇的顏色很有限，預設的白色烤漆是基本款，黑、灰、藍需要另外加價 37,500 元新台幣，紅色又更特別，需要額外支付 75,000 元才能擁有。不過，也有許多車主不想要受限於原廠，費盡心思替自己的愛車重新換裝，粉紅色就是熱門選項之一。

▲ 粉紅色的 Model X 在路上肯定讓你回頭多看一眼。（Source：TeslaMotorsClub）

來自芝加哥的 Verity 是熱愛粉紅色的男人，夢想之一就是擁有一台粉紅色跑車，Model X 意外的滿足了他對跑車的追求與實用度，而入手 Model X 後他就花了大半年時間研究怎麼幫它變身。

最終成果讓人驚豔，更特別的是這身粉紅外衣，真的是外衣。「雖然我很衝動，但我的理智告訴我，如果有一天我要賣掉這輛車，粉紅色烤漆恐怕會是個問題。」Verity 表示，他最後找到一種越野賽車常用的「噴膜」（Plastic Dip），可以覆蓋車身，又能快速移除。

另一位熱愛粉紅色的女網友，則是選擇用包膜方式，將她的紅色 Model 3 變成粉紅色（既然如此，何必多花錢買紅色的呢？）

▲ 身為刺青師的 Bratty Girl 選擇了粉紅色包膜來變身。（Source：BrattyGirl）

雖然名稱一樣是粉紅色，還是有差別，個人比較喜歡 Model X 那種朦朧粉，但這也跟選擇方式有關，由於這輛 Model 3 原來是紅色，包膜之後可能會讓顏色較偏紅；Model X 用噴膜方式，且車身是白色，因此成果較淺。

美國知名彩妝部落客 Jeffree Star 也有一輛粉紅色 Model X，對熱愛粉紅色的他來說，這顏色似乎再正常不過了。

不過對車廠來說，粉紅色還是比較小眾的配色，且濃淡程度影響很大，要看到原廠提供粉紅色選項恐怕不容易。

看完這幾輛粉紅色特斯拉，不禁讓人想像一下 Cybertruck 交車後，會變出什麼超酷配色。如果你也在路上看過顏色獨特的車輛，歡迎留言跟我們分享。

（首圖來源：TeslaMotorsClub）

粉紅色口罩行動發酵 政府機關紛換臉書大頭貼

許多政府機關今天臉書紛紛換上粉色系大頭貼，國發會寫下：「顏色是沒有性別界線的，溫暖、明亮的粉色代表正向積極的國家前進力量」；衛福部說：「衛福編編跟總柴都希望，這世界上的所有人都能擇其所愛、愛自己、肯認自己、互相尊重，自由自在的健康生活！」教育部的臉書則以一句簡單的「粉紅色就萌」並配上可愛的表情符號。
