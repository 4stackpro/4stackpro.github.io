澳洲男孩名叫Corona遭霸凌 湯姆漢克斯贈同名打字機

澳洲電視台報導，澳洲一位男孩因為名叫Corona遭到霸凌，寫信告訴美國影星湯姆漢克斯自身遭遇。湯姆漢克斯親自打了一封回函並寄送一台Corona牌古董打字機給他，作為鼓勵。

湯姆漢克斯（Tom Hanks）與妻子麗塔威爾森（Rita Wilson），上月在澳洲昆士蘭州（Queensland）黃金海岸（Gold Coast）確診感染新冠肺炎後，在當地隔離逾兩週。

湯姆漢克斯夫婦返回美國後，住在黃金海岸的8歲男孩柯羅納．戴．伏里斯（Corona De Vries）於本月初寫信給兩人。

澳洲第七頻道（Channel 7）報導，男孩在信中寫道：「我在新聞上聽到你和妻子染上冠狀病毒。你們還好嗎？」

男童還寫道：「我喜愛自己的名字，但學校的人叫我冠狀病毒，當人們這樣叫我時，我感到非常傷心及憤怒。」

為此，湯姆漢克斯特地使用他帶去黃金海岸的Corona牌古董打字機打回信。他在信中寫道：「你的信讓我和妻子感到相當窩心。」

他在信中告訴男孩：「你知道嗎？你是我認識的人當中唯一名叫Corona的人，就像是太陽四周的日冕，有如一項皇冠。」

Corona在英文及拉丁文中意指冠或日冕，冠狀病毒名為Coronavirus，因為在電子顯微鏡下可看到病毒表面類似皇冠上的突起。

湯姆漢克斯寄給男孩一台Corona牌打字機，並在信中寫道：「我想這台打字機將會適合你。問大人要如何操作。用它來打字回信給我。」他還在結尾時寫道：「附註，你已成為我的朋友。」

湯姆漢克斯熱愛收集打字機，旅行時總愛帶著Corona牌打字機。他於澳洲染疫期間，曾在推特上傳這款打字機的照片，並戲稱這「曾經是我的最愛」。

Joao Corona 是《英雄聯盟 》(League of Legends) 的老玩家，而他的姓氏「Corona」一直以來都沒有什麼問題。但就在本月初，當他登入的時候，赫然發現 Riot Games 官方要求他必須更改「Corona」這個他在遊戲中使用已久的 ID。

在這場疫情失控之前，人們對於「Corona」的印象幾乎都是來自於墨西哥知名啤酒品牌「可樂娜啤酒」。這個詞源自於西班牙文，也就是「皇冠」的意思。如今，「Corona」卻成為了一個讓全球民眾聞之色變的致命病毒名稱。這對於 24 歲的 Joao Corona 也帶來了完全沒有料想到的影響，而他在《英雄聯盟》中使用「Corona」這個 ID 也已經有 1 整年的時間了。

在今年 4 月 1 日，當他登入《英雄聯盟》的時候，卻發現來自官方的通知，要求他更改遊戲 ID，不然就無法繼續使用該帳號遊玩。

Joao Corona 在《英雄聯盟》(League of Legends) 的 Reddit 討論版PO文表示「一開始我還覺得挺有趣的，因為最近在玩的時候，不少玩家都會稱讚我取了一個『好名字』，但我沒太在乎，之後回信給 Riot Games，希望他們能夠了解這是個有趣的誤會，而且我的ID應該要維持在白名單中。」

facebook

Joao Corona 試圖在一封客服郵件中向 Riot Games 解釋一切的來龍去脈，並公開了這封郵件的截圖。他是一位從 2014 年就開始遊玩《英雄聯盟》的休閒玩家，他最初所使用的 ID 為「Squid Corona」，大約在一年前，他將 ID 改成了「Corona」，他只是要讓朋友更容易在遊戲中找到他。

然而，這說法卻無法說服《英雄聯盟》的遊戲客服。「懇請您將在遊戲中的 ID 改成其他你所喜愛的名稱」名為 Monocromie 的客服如此回應「與此同時，我也會試試看是否能夠讓您保有目前的 ID」

接著Joao Corona 繼續上訴，但 Riot Games 依然回絕了他的要求。主要是線上遊戲中環境不夠友善，有太多 ID 常常被玩家拿來惡意嘲弄。其他玩家也在遊戲中發現了許多遠比「Corona」更加明顯且充滿惡意的 ID，像是「CoronaComin4U」（冠狀病毒查水表） 或是「ChinaVrs」

（中國病毒） 等。從這可以發現，可能只是官方在尋找各種與疫情有關的惡意 ID 時，剛好偵測到「Corona」而已。而對這個事件，Riot Games 並沒有立刻做出回應，而是在之後給予 Joao Corona 答覆。

「我們並不認為強迫玩家在這種情況下改名符合我們的政策，所以我們將會持續深入探討。」《英雄聯盟》公關負責人 Ryan Rigney 在 Reddit 論壇上回應。目前，Joao Corona 也已經順利取回了自己的 ID。

「感覺真的很蠢。」Joao Corona 在接受Buzzfeed訪談時表示「我確實能夠理解為何某些人會對此感到不太高興，但好笑的是，如果你什麼都要控制，那肯定會引起大家不爽的」
