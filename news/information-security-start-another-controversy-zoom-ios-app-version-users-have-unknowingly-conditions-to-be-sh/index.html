再掀資安爭議！Zoom iOS App 版用戶曾在不知情狀況下「被分享」資訊給臉書

在新冠肺炎疫情之下，遠端工作、會議需求暴增，視訊軟體 Zoom 最近變成當紅討論話題，但同時也被發現存在不少資安問題，去年曾被爆出 Mac 用戶會被強制加入會議並開啟鏡頭（現在已修復）之外，最近又爆發新一波資安爭議。

外媒 Motherboard 發現，Zoom 的 iOS App，會自動將使用者的數據，包括所在市區、城市和裝置詳情，透過 Graph API 發送給臉書 Facebook，即使用戶沒有 Facebook 帳戶。

雖然很多 APP 在開發過程，會使用臉書的開發工具（SDK），會傳送資料給臉書算是正常。但重點就在 Zoom 的隱私條款中，沒有清楚註明這點，只有寫道「我們的第三方服務提供者和廣告夥伴，例如 Google Ads 和 Google Analytics，會在您使用時搜集相關資料」，裡頭並沒有提及在沒有臉書帳號的狀況下，也會傳送資料給臉書。

這個問題被爆出來之後，Zoom 官方先是道歉，表明會盡快修正，也更新了 iOS App，讓用戶打開 App 的時候，數據不會再分享到臉書。

Zoom 解釋，他們在 APP 中利用 SDK 加入臉書登入的功能，但沒注意到背後會有數據分享的問題，新 APP 版本會把 SDK 拿掉，用戶還是可以用臉書登入，但會改成導到瀏覽器登入，功能上不會有變化，新的版本也已經放上 App Store 給用戶下載。

遭盜6.4萬美元 婦女疑用Zoom視頻開會導致

遭盜6.4萬美元 婦女疑用Zoom視頻開會導致

更新 2020-04-17 7:10 AM

【大紀元2020年04月17日訊】一名比弗利格林（Beverly Glen）女子布朗（Marla Brown）近日向洛杉磯警方報案，懷疑一名駭客利用視頻會議APP「Zoom」，遠程侵入了她的電腦，從她的銀行帳戶盜走了超過6萬美元。

如今，洛杉磯警局和Zoom公司都在調查此事，Zoom公司說，還沒有證據顯示，該公司的軟件可能要對此事負責。

據FOX11報導，布朗幾天前查看自己的銀行帳戶時，震驚地發現了幾筆未經她授權的大額轉帳，轉出時間就在她下載了Zoom APP之後。

她說，先是轉走了5萬美元，兩天後又轉走了1萬多。最終她被盜走了約6.4萬美元。

實際上，在發現帳戶被盜前，布朗已經從電腦上刪除了Zoom。布朗是在3月底下載Zoom的，以便在疫情期間與同事進行視頻會議。但下載兩天後（4月初），她注意到一些不尋常的事情。

一次她準備打開計算機，發現電腦桌面頂部有一個小小的光標，她仔細查看，該光標說正接受另一個地點的操作。震驚之餘，她立即刪掉了Zoom。

布朗之前也聽說了一些關於Zoom泄漏用戶個資的傳聞，但是她想，她就用了這一次，應該沒什麼問題。可幾天前，她查看銀行帳戶發現被盜。布朗當時立即打電話給銀行，並向洛杉磯警局報案。警局證實，已經啟動了一個身分盜竊調查，但沒有進一步置評。

Zoom公司的聲明說：該公司正在調查這個事件，但他們迄今還沒有發現證據，這起電腦入侵與Zoom的合法軟件有關。Zoom用戶應當清楚，Zoom平臺的合法域名只有zoom.us、zoom.com 和zoom.com.cn，而其它的域名都不屬於Zoom公司。請用戶在點擊鏈接前仔細查看URL，注意那些很相似的域名以及拼寫錯誤。

但布朗說，她是直接從合法的網站zoom.us下載的Zoom APP。

最近，多家媒體報導Zoom APP存在重大安全漏洞，用戶在 iOS系統中使用Zoom時，該App會將用戶位置等個人資訊發送給臉書，即使該用戶沒有臉書帳號也一樣。

另據Bleeping Computer的報告披露，超過50萬個 Zoom帳戶在黑市網絡被賤價出售，有時甚至是免費。

針對相關報導和傳聞，Zoom公司回應說：為消費者服務的Web服務通常會受到此類活動的攻擊，一般都是利用其它平臺泄露的大量數據，測試用戶是否在別處重複使用這些數據。這種攻擊通常不會影響Zoom的大企業用戶，他們的使用者僅採用自己的單邊登錄系統。

Zoom並表示已經僱用情報公司進行調查，包括一家已關閉數千網站的公司，其目的是誘騙用戶下載惡意軟件或泄露密碼等信息。

不過布朗依舊表示，雖然她也認為Zoom的功能不錯，但她仍然懷疑Zoom為入侵她帳戶的駭客打開了門。布朗已經要求銀行關閉她的帳戶，並已啟動了追回被盜存款的程序。◇
