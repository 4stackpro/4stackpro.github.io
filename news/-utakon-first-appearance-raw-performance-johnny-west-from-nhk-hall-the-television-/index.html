＜うたコン＞初登場！ジャニーズWESTがNHKホールから生パフォーマンス（ザテレビジョン）

“明日の「うたコン」にクミコ＆井上芳雄、さだまさし、真田ナオキ、細川たかしら”

6月23日(火) 午後7時57分からNHK総合で生放送の「うたコン」にクミコ＆井上芳雄、さだまさし、真田ナオキ、細川たかしらが出演する。

明日も生放送でリクエストに応え、さだまさしは夫婦の名曲と、大黒摩季は元気が出る大ヒット曲を。細川たかしは朝ドラ「エール」ゆかりの歌を熱唱。初登場のジャニーズWESTはメンバーそろってNHKホールで生パフォーマンス。「父の日」に送る歌、ファンモン、島津亜矢などの貴重映像も。クミコ＆井上芳雄はつんく♂作の話題曲。さらにハスキーボイスで注目の真田ナオキなど。

「うたコン 〜わたしたちには歌がある〜」

放送：NHK総合

日時：6月23日(火) 夜7時57分〜8時42分

出演：井上芳雄 / 大黒摩季 / クミコ / さだまさし / 真田ナオキ / ジャニーズWEST / 細川たかし / フラッシュ金子 / MUSIC CONCERTO

司会：谷原章介 / 赤木野々花

“次回「うたコン」は70日ぶりNHKホールから生放送、野口五郎、天童よしみ、氷川きよし、辰巳ゆうとら”

6月9日(火) 午後7時57分からNHK総合で生放送の「うたコン」は3月31日以来70日ぶりにNHKホールから生放送、野口五郎、天童よしみ、氷川きよし、辰巳ゆうとらが出演する。

新型コロナウイルス感染拡大防止の観点から、観客を招く形では実施できない状況ではあるが、久しぶりにNHKホールから、「うたコン」ならではのスペクタクルなステージショーを生放送で送る。氷川きよし熱唱で限界突破＆朝ドラ「エール」で話題のアノ歌も披露。デビュー50周年の野口五郎は、盟友・西城秀樹との秘蔵映像＆コブクロ小渕が手がけたバラード披露。西川貴教は昭和歌謡ヒット＆注目の新ユニットで登場。天童よしみ・辰巳ゆうとの情熱ソング・リクエストに応え、竹内まりや、井上芳雄＆山崎育三郎らミュージカルスター豪華共演の貴重映像まで。

MC谷原章介コメント

医療関係者、そして皆様の努力のおかげで、新型コロナウイルスの感染拡大も徐々に収束してきましたが、まだ終わりではありません。

こんな時だからこそ歌の力でテレビをご覧の皆様に、癒しを、喜びを届けられるよう私たちも一歩一歩進んでまいります。そしてこの先、NHKホールで皆様をお迎えできる日に向けて頑張ります！

「うたコン 〜わたしたちには歌がある〜」

放送：NHK総合

日時：6月9日(火) 夜7時57分〜8時42分

出演：辰巳ゆうと / 天童よしみ / 西川貴教+ASCA / 野口五郎 / 氷川きよし

司会：谷原章介 / 赤木野々花

“井上芳雄・山崎育三郎ら、今夜放送の「うたコン」にVTR出演”

本日6月9日19:57からNHK総合で放送される「『うたコン』わたしたちには歌がある！」に 井上芳雄、 山崎育三郎らがVTR出演する。

「うたコン」は演歌や歌謡曲、ポップス、洋楽、クラシックなど多彩なジャンルの音楽を視聴者に届ける番組。70日ぶりに東京・NHKホールから生中継される今回の「うたコン」では、視聴者リクエストに応えて井上、山崎らの映像が放送される。

本番組には司会を務める谷原章介と赤木野々花のほか、辰巳ゆうと、天童よしみ、西川貴教＋ASCA、野口五郎、氷川きよしが登場予定だ。
