【專訪】〈How To Be Lonely〉意外成防疫歌單！Rita Ora「學習獨處才能找到力量」

在去年客串《名偵探皮卡丘》（Detective Pikachu）、與Kygo共譜主題曲〈Carry On〉後，芮塔歐拉（Rita Ora）除了發行一首與Tiësto和強納斯布魯（Jonas Blue）合作的〈Ritual〉，其他時間不是在《英國蒙面歌手》（The Masked Singer UK）擔任評審，就是在錄音室籌備第三張專輯。如今終於等到首波主打〈How To Be Lonely〉發行，KKBOX特別與芮塔越洋通話，問問她對於新篇章的期待。

Lewis Capaldi助陣譜寫！要你學會如何獨處

新單曲〈How To Be Lonely〉最大的賣點，除了是芮塔近一年以來、開啟新時代的首支單曲，她更找來了英倫超強新人路易斯卡柏狄（Lewis Capaldi）共同創作，芮塔表示，兩人當初會搭上線，是因為巡迴時共用同一位攝影師，因此結識成為好友。「從路易斯出道之後，我就一直是他的粉絲，他人很好，我們聊了一段時間決定要一起寫一首歌，〈How To Be Lonely〉是這樣來的！」

〈How To Be Lonely〉講述一段感情破裂後，剛恢復單身的人要如何與自己獨處。芮塔說到，現在年輕一代的人似乎都不甘寂寞，不會問問自己到底需要什麼，反而弄得全身傷，她想藉由這首歌告訴大家，寂寞不代表脆弱，因為「學會與自己相處，才能真正找到心中源源不絕的力量。」

MV即將上線！著重分手後的情緒階段變化

在 Instagram 查看這則貼文 RITA ORA（@ritaora）分享的貼文 於 PDT 2020 年 3月 月 13 日 上午 5:00 張貼

縱使現在全球因為疫情的關係，許多釋出和發行都因此延期，但〈How To Be Lonely〉的MV在芮塔歐拉的擔保下，很快就會與大家見面了！「這次音樂錄影帶的構想是，將一個人分手後歷經的情緒變化，用不同意象和情景表現出來。從對外界一切都非常敏感，或是容易因為小事而動怒、難過，到最後浸泡在黃金粉裡，代表到達豁然開朗的階段。」芮塔還興奮地說到，她在MV中會衝刺撞破高牆，還會打扮成外星人的模樣，「因為單身的時候會讓一個人看事情的角度瞬間不同，就好像來到一顆奇異星球上。」

至於大家期待已久的新專輯，芮塔表示現在還在準備階段，會非常仔細去挑選想要合作的人選和歌曲，也希望在今年還會有更多單曲的釋出。「我的目標就是希望大家聽到我的音樂能開心，開心到手舞足蹈的那種，這也是為什麼〈How To Be Lonely〉會釋出舞蹈版的音樂錄影帶。音樂與人的連結是非常重要的，在這種時候我們更需要音樂來度過難關。」

想接《蒙面歌手》第二季！對正在防疫的大家喊話

談到《英國蒙面歌手》，芮塔毫不猶豫表示自己愛死這個節目，她很享受在音樂節目擔任評審！「我做過《英國好聲音》（The Voice UK）、《The X Factor》的評委，還擔任過MTV EMA的主持人，我很愛任何與音樂相關的電視節目，能夠聽到這些有才華的歌手被挖掘、培養，這是最令人開心的事！」另外，芮塔表示當初會接下《英國蒙面歌手》，主要是因為她太容易「以為自己認得這個聲音」，「他們拿下面具的那一刻真的太令人興奮！」同時透露非常有意願接下第二季。

訪談最後，芮塔歐拉語重心長向所有人喊話：「請大家盡量配合自己國家的防疫規則，記得勤洗手、沒事不要出門。」芮塔也表示看到很多粉絲拿〈How To Be Lonely〉當「隔離歌曲」，心中五味雜陳：「這一切太過巧合了！沒想到現在真的是大家學習獨處的時候，只願大家平安，聽著我的歌在家運動跳舞。」
